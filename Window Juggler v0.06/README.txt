
WindowJuggler plugin by EsseEmmee
for OllyDdg 1.08 and greater


----------------------------------------

INSTALLATION:
Place the DLL in the plugin folder. You can see the currently configure folder and eventually change it by clicking on the "Options->Appearance" menu item and by clicking on the "Directories" tab.

----------------------------------------

USE OF THE PLUGIN:
By clicking on the "Plugins->WindowJuggler" menu item you open up the plugin dialog box. Pressing the SHIFT key while over a window will let you capture a few information about that window and afterwards will enable you to send it a few messages or simulate certain actions.

----------------------------------------

NOTICE: The plugin has been tested only on a Windows 2000 machine.
