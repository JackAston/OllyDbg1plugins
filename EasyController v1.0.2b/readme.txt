EasyController by esso_x
http://www.at4re.com/

Use this plugin to help quickly control actions and operations within OllyDbg. Some of these operations include:

- remove all breakpoints and save to a backup file
- restore all breakpoints from a backup file to prevent losing your breakpoint history
- go to the start of a procedure
- go to the end of a procedure
- buttons to quickly open; Calculator, Notepad and RegEditor
- a quick one button UDD and BAK cleaner operation
- search button for string references inside the debuggee
- search button for all Call references
- quickly open the debuggee application data folder
- reminds you in case the date and time is changed
- quick screen capture to a default "images/" directory
