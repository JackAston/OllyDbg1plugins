File Name: oDump.dll
File Submitter: waliedassar

You can use this plugin within OllyDbg v1.10 as a replacement for OllyDump. Many anti-dumping tricks that OllyDump has suffered from have been bypassed in oDump.

N.B. Still under development to defeat more anti-dumping tricks.
